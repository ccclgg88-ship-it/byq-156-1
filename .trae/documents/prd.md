## 1. 产品概述

「压力气泡消消乐」是一款圆嘟嘟宠物主题的三消休闲小游戏，玩家通过交换相邻气泡形成三个及以上同色连线进行消除，与可爱的宠物伙伴一起释放压力并积累「轻松值」。
- 目标用户：寻求轻松解压体验的休闲游戏玩家
- 核心价值：将经典三消玩法与萌宠互动结合，打造治愈系游戏体验

## 2. 核心功能

### 2.2 功能模块

1. **选关地图页**：关卡路径展示、星级显示、进度追踪
2. **游戏棋盘页**：8×8 气泡棋盘、交换消除、特殊气泡、步数/目标面板
3. **宠物助威区**：宠物角色展示、cheer 动画、台词气泡、大招按钮
4. **结算弹窗**：星级评定、轻松值统计、奖励展示、下一关/重试

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 选关地图页 | 关卡路径 | 曲线/蜿蜒路径展示15+关卡节点，已通关显示星级，未解锁灰色 |
| 选关地图页 | 顶部状态栏 | 总轻松值、宠物头像、设置按钮 |
| 游戏棋盘页 | 8×8 棋盘 | Canvas 渲染气泡棋盘，支持点击选中和交换操作 |
| 游戏棋盘页 | 步数/目标面板 | 显示剩余步数、当前关卡目标进度（如消除30个红色气泡） |
| 游戏棋盘页 | 特殊气泡效果 | 四消/五消/L-T形消除生成特殊气泡，触发行清/列清/全色爆炸 |
| 游戏棋盘页 | 轻松值条 | 显示轻松值累积进度，满槽后可触发宠物大招 |
| 宠物助威区 | 宠物角色 | 圆嘟嘟宠物站在棋盘旁，不同状态切换表情和动画 |
| 宠物助威区 | 台词气泡 | 随机显示鼓励短句（如"加油鸭！""你可以的！"） |
| 宠物助威区 | 大招按钮 | 轻松值满槽后激活，点击触发随机清除一种颜色全部气泡 |
| 结算弹窗 | 星级评定 | 1星通关、2星剩余步数≥5、3星剩余步数≥10且无无效交换 |
| 结算弹窗 | 统计信息 | 本关消除数量、连消次数、获得轻松值 |
| 结算弹窗 | 操作按钮 | 下一关、重试、返回地图 |

## 3. 核心流程

用户打开游戏 → 进入选关地图 → 选择关卡 → 进入游戏棋盘 → 交换气泡消除 → 连锁消除积累轻松值 → 达成目标或步数耗尽 → 结算弹窗 → 下一关或重试

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：柔和的薰衣草紫 (#B8A9E8) 作为主色，配合温暖奶油白 (#FFF8EE) 背景
- **辅色**：珊瑚粉 (#FF9B9B)、薄荷绿 (#7DCEA0)、天蓝 (#85C1E9)、向日葵黄 (#F9E154)
- **气泡颜色**：猫爪粉 (#FF9B9B)、狗骨棕 (#C4956A)、猪鼻桃 (#FFB7C5)、奶鱼蓝 (#85C1E9)、幸运草绿 (#7DCEA0)、阳光橙 (#FFB347)
- **按钮风格**：圆角胶囊按钮，带柔和阴影和按压缩放动效
- **字体**：圆润可爱的中文字体（如站酷快乐体），数字使用圆润风格
- **布局风格**：游戏区域居中，宠物在右侧/下方，信息面板在顶部
- **图标/Emoji**：使用宠物元素符号 🐱🐶🐷🐟🍀🍊 代表不同颜色气泡

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|

### 4.3 响应式设计

- 桌面优先设计，游戏区域固定宽高比
- 棋盘尺寸自适应：根据视口大小缩放 Canvas
- 移动端：宠物区域移至棋盘下方，触控适配（touch 事件）
- 最小支持 375px 宽度

### 4.4 关卡配置

关卡数据以 JSON 格式驱动，包含：
- `id`：关卡编号
- `moves`：步数限制
- `targets`：目标数组（类型、颜色、数量）
- `obstacles`：障碍物位置数组（不可移动格）
- `bonusMoves`：奖励步数（+5 奖励关）
- `colors`：本关使用的气泡颜色数量（前期4色，后期5-6色）
- `shuffleLimit`：洗牌次数限制
